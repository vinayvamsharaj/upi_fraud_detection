About This Project

This mini project predicts whether a UPI transaction is fraudulent using a machine learning model. The web application is built using Flask and a trained scikit-learn model saved as model.pkl.

Tech Stack

  • Python • Flask
  • scikit-learn • joblib
  • SQLite • pandas
  • HTML • CSS • JavaScript

Input Features

  • Amount
  • Hour of Transaction
  • New/Old Device
  • Transactions Count Today

How It Works

  1. User enters transaction details
  2. Model predicts fraud (0/1)
  3. App shows result + probability meter
  4. History saved into SQLite
